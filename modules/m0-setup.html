Ambiente de trabalho

O terminal à direita roda CPython 3.11 em WebAssembly. Para o seu ambiente local:

BASH# Verificar versão
python --version

# Criar e ativar venv
python -m venv .venv
source .venv/bin/activate    # Linux/macOS
.venv\Scripts\activate       # Windows

# Stack de segurança
pip install requests aiohttp pydantic pandas openpyxl \
    scapy psutil cryptography paramiko pyyaml bandit

pip freeze > requirements.txt
⚠ NUNCA COMMITE .venv/ e .env. API keys em variáveis de ambiente, nunca hardcoded.

Padrões essenciais

1. Versão do Python

PYTHONimport sys
sys.version

2. Plataforma de execução

PYTHONimport platform
platform.platform()

3. List comprehensions — limpeza de IOCs

PYTHON[x*2 for x in range(5)]

# aplicação real — limpar lista de IPs com espaços
ips = ["192.168.1.1 ", " 10.0.0.5", ""]
[ip.strip() for ip in ips if ip.strip()]

4. Dict.get com default — APIs sempre retornam dict

PYTHONresp = {"malicious": 5, "suspicious": 2}
resp.get("malicious", 0)
resp.get("undetected", 0)  # default seguro

5. Tratamento de erro inline — APIs falham, scripts não podem cair

Em script .py normal você usa try/except em bloco. No REPL, uma técnica equivalente em uma linha é o conditional expression: 

PYTHON# Bloco tradicional (em script)
try:
    result = int("abc")
except ValueError:
    result = "ok"

# Inline com list comprehension — útil pra parsing em massa
entradas = ["42", "abc", "7", "x"]
[int(x) if x.isdigit() else "skip" for x in entradas]
